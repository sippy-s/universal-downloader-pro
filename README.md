# Universal Downloader Pro (UDP) 🚀

یک ابزار قدرتمند در GitHub Actions برای دانلود هر فایلی (مستقیم یا یوتیوب) و تبدیل خودکار به قطعات ۹۰ مگابایتی ZIP، کاملاً رایگان و بدون نیاز به نصب.

## ✨ ویژگی‌ها

- دانلود لینک مستقیم و ویدیوهای یوتیوب
- تقسیم خودکار فایل به قطعات ۹۰ مگابایتی (برای عبور از محدودیت گیت‌هاب)
- قابلیت رمزگذاری ZIP
- پشتیبانی از چند لینک همزمان
- تشخیص خودکار نوع لینک (یوتیوب یا معمولی)
- ذخیره دائمی فایل‌ها در ریپازیتوری خودتان

## 🎬 نحوه استفاده

1. به تب **Actions** بروید.
2. در سمت چپ، روی **Universal Downloader Pro (UDP)** کلیک کنید.
3. دکمه **Run workflow** را بزنید.
4. لینک(ها) را وارد کنید (برای چند لینک با فاصله جدا کنید).
5. کیفیت (برای یوتیوب) و رمز (اختیاری) را تنظیم کنید.
6. **Run workflow** را بزنید.
7. منتظر بمانید تا کار آن تمام شود (۵-۱۰ دقیقه بسته به حجم).
8. پس از اتمام، به تب **Code** بروید و داخل پوشه `final_zips` فایل‌های `.z01`, `.z02`, ... و `.zip` را دانلود کنید.
9. برای استخراج فایل اصلی، همه قطعات را در یک پوشه قرار دهید و با 7-Zip یا WinRAR فایل `.zip` را Extract کنید (در صورت وجود رمز، آن را وارد کنید).

## ⚠️ نکات مهم

- حجم کل فایل نهایی نباید از **۲ گیگابایت** بیشتر شود (محدودیت گیت‌هاب برای ریپازیتوری).
- برای دانلود فایل‌های بزرگتر از ۲ گیگ، از سرویس‌های ابری واقعی استفاده کنید.
- این ابزار فقط برای استفاده شخصی و آموزشی توصیه می‌شود.
- لطفاً از دانلود مطالب دارای کپی‌رایت خودداری کنید.

## 🛠️ توسعه

این پروژه از `yt-dlp` و `aria2` استفاده می‌کند. شما می‌توانید کد را در فایل `.github/workflows/download.yml` سفارشی کنید.
